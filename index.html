Bot Commands

This project page gives a description of the current command list of Bellematon.

Basic commands

  • about - Basic bot info
  • guide - Get a link to this guide
  • say [text] - Force to say something
  • set [command], rank - Sets a command to cetain rank.
  • set mod, [stretching|flooding|caps], [on|off] - Sets moderation level when bot is mod
  • addcom [command],[rank],[output] - Add a custom room only command.
  • delcom [command] - Delete a custom command.
  • comlist - Lists the command list.
  • autores [add|del|list] - Automated response for certain text patterns.
  • addjoinphrase [user], message - Sets a join phrase.
  • deljoinphrase [user] - Deletes a join phrase for the user.
  • editjoinphrase [user], message - Edits the set joinphrase.
  • viewjoinphrases - Views all the joinphrases for the given room.
  • addquote - Adds a quote in the room quotelist.
  • delquote - Deletes the given quote from the room quotelist.
  • quotelist - Shows the quotelist for the given room.
  • quote - Shows a random quote or based on the given argument.

Custom commands help

  • {arg} - Takes input from the custom command when added to the output.
  • {by} - Adds the user's username to the command output when used in output.
  • {pick} - picks one of the args.
  • {rand[number]} - throws a random number from 1 to the number.
  • {choose[choice1, choice2, choice3, etc]}- chooses between choices.

Miscellaneous commands

  • meme - Random Lenny
  • joke - Random Chuck Norris joke
  • rk - Kicks the given user from the room, requires bot to be mod.
  • pair - Pairs the user with the argument
  • ship - Pairs the two arguments.
  • rank - Shows the rank of the user or the argument, won't work if the ladder rating is too long to be displayed.
  • age - Shows the age of the user or the argument based on registration.
  • regdate - Shows the registration date for the user or the argument.
  • 8ball - Random response over a question
  • usage - Usagestat command.

Games

  • signups - Starts the game in the argument [trivia/hangman/anagrams/blackjack/statspread/kunc/passthebomb/ambush/crazyeights]
  • g - used for guessing
  • atm - lists your points on the leaderboard.
  • autostart - sets the autostart timer on a given game in seconds.
  • start - Starts the given game.
  • end - Ends the game prematurely.
  • join - Registers you for an ongoing game.
  • leave - Deregisters you from the players.
  • pass - Passes the bomb in pass the bomb.
  • hit/stay - Hit or stay in Blackjack.
  • play - Plays a card in crazyeights.
  • fire - Snipes the argument in Ambush.
  • randgame - Plays a random game.
